WH

Website Helper

Generate themes and paste-ready snippets for your Odoo pages.

Generate Theme
Describe the look; get a themed CSS block with nav/pill tokens.
If left as-is, a generated CSS file will be written under static/website_helper/ when you toggle apply.
Inject Custom Theme
Paste CSS and write it into a target file inside the repo.
Path is relative to the repo. Use append mode to keep existing styles.
Generate Website Layout
Creates a copy/paste HTML + CSS block you can drop into an Odoo snippet.
Leave blank to auto-pick from your brief (teal/blue/purple/etc.).
Adds a cover image blended behind the layout.
Scripts are stripped. If empty, a default layout is generated.
